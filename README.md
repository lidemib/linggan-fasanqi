# 灵感发散器 🥟

> 今天吃什么？一键搞定！

## 项目简介

**灵感发散器**是一款帮助用户快速决策"今天吃什么"的轻量级 H5 应用，通过三级发散选品逻辑（大品类 → 细品类 → 附近商家），结合优惠券领取和到店核销，打通"选餐决策"到"到店消费"的完整闭环。

**核心场景：** 用户中午不知道吃什么 → 打开小程序 → 三步选出想吃的 → 领券 → 进店消费 → 商家核销

## 本地运行
